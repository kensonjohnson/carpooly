### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/codewizard13/udd-group-project-carpooly.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the develpment server
   ```
   npm run dev
   ```
