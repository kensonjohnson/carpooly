<br />

## Installation

<br />

Clone the repo:

```sh
git clone https://github.com/kensonjohnson/carpooly.git
```

<br />

### Web Instructions

<br />

1. You'll need node version 18 or higher.

```sh
   npm install npm@latest -g
```

2. Move to the Web Project folder.

```sh
   cd web
```

3. Install dependencies.

```sh
   npm install
```

4. Run the develpment server

```
   npm run dev
```
